# Presenter notes — Scenario overview

**Time:** ~20s · **Total deck target:** ~3 min spoken

## Say

> "Every business wants this: an analyst asks an AI agent for a sales chart and
> just gets one. No SQL, no notebook. Useful. But _useful_ turns risky when the
> chart code runs right next to the database credentials. The fix is to move that
> code into an isolated sandbox. Three quick demos: the normal request, the
> attack, and the fix."

## Do / cues

- Press **Space** to reveal **Useful → Risky → Safer** in time with the line.
- Don't over-explain here; this is the 20-second frame for the whole talk.
